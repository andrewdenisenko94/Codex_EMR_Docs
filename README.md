# CHOP Urology NoteForge

Markdown-first resident documentation workspace for pediatric/urology note drafting, synthetic examples, and QA checks.

## Structure
- `docs/`: style, schema, QA, registry
- `templates/`: paste-safe note templates
- `examples/`: synthetic raw input, parsed objects, outputs, QA
- `validators/`: markdown specs for validation checks
- `private/`, `scratch/`: local-only PHI/scratch (gitignored)

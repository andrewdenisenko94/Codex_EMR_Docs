# QA Checklist

- [ ] Laterality consistent across HPI, imaging, assessment, and plan.
- [ ] No unsupported voiding claim; PVR/emptying source documented.
- [ ] Device consistency (Foley/SPT/PCN/stent/vesicostomy).
- [ ] UA/UCx relationship addressed (infection vs colonization).
- [ ] Red flags: fever/sepsis + obstruction/hydro/stone + renal dysfunction.
- [ ] Key values preserved (date, Cr/WBC/Hgb, PVR, size, catheter French).
- [ ] Lab shorthand style correct for output type.
- [ ] Bullet style correct for output type.
- [ ] CHOP objective-data dated-line format preserved.
- [ ] Disposition supported by objective data.
- [ ] Follow-up owner and timing specified when needed.
- [ ] No PHI in examples.

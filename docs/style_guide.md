# Style Guide

- Voice: clinical, direct, compressed, resident-style.
- Never invent facts; preserve dates/values/laterality/device sizes.
- CHOP consult objective data uses dated single-line entries:
  - `MM/DD/YY Labs: WBC, Hgb, Cr`
  - `MM/DD/YY UA: ...`
  - `MM/DD/YY UCx: ...`
  - `MM/DD/YY Imaging: ...`
- Bullets only in Plan unless explicitly requested.
- Plan bullets use `-` hyphens; EPIC handoff plan uses `[]` checklist.
- No arrows; ASCII-safe text only.
- In consult notes use Cr/WBC/Hgb; in list format use C/W/H.
- Keep non-GU bloat out unless directly relevant to GU decisions.
